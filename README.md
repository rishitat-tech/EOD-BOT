# EOD Slack Bot

Google Apps Script based EOD bot for Slack.

## What it does

- Posts EOD reminder using Google Form link
- Collects EOD updates through Google Forms
- Posts daily summaries to Slack
- Supports separate Main/Lab summary and Ego summary
- Runs using Apps Script triggers
- Does not require laptop/server to stay on

## Schedule

- Reminder: 6 PM to 7 PM
- Summary: 9 PM to 10 PM

## Channels

- Main EOD: C0ALYC9T0N5
- Ego EOD: C0AKFH5A406
- Lab summary: C0AVBEWQX7T

## Required Apps Script Property

- SLACK_BOT_TOKEN

## Important

Do not commit Slack tokens or secrets.
